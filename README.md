# Landing Page - Christophe Seyler

Bienvenue sur le projet de la landing page de Christophe Seyler. Ce projet est conçu pour présenter les compétences, les projets et les intérêts de Christophe dans un format moderne et attrayant.

## 🚀 Fonctionnalités

- **Présentation personnelle** : Une introduction claire et concise.
- **Compétences principales** : Affichage des compétences clés alignées horizontalement.
- **Projets** : Section dédiée aux projets réalisés.
- **Design moderne** : Utilisation de TailwindCSS pour un style réactif et élégant.
- **Mode sombre** : Support du mode sombre pour une meilleure accessibilité.

## 🛠️ Technologies utilisées

- **React** : Framework JavaScript pour la construction de l'interface utilisateur.
- **TailwindCSS** : Framework CSS pour un design rapide et réactif.
- **Vite** : Outil de build rapide pour les projets modernes.

## 📦 Installation

1. Clonez le dépôt :

   ```bash
   git clone <url-du-repo>
   ```

2. Accédez au dossier du projet :

   ```bash
   cd landindpage
   ```

3. Installez les dépendances :

   ```bash
   npm install
   ```

4. Lancez le serveur de développement :

   ```bash
   npm run dev
   ```
